# Vec3 详细接口文档

## 概述

`Vec3` 是 `mine.math` 模块的 **三维浮点向量** 类型,表示 3D 空间中的向量、位置或方向。作为 POD 类型,`Vec3` 支持编译期常量、高效内存布局,适用于矩阵变换和三维几何计算。

**核心特性:**
- **POD 类型**:Plain Old Data,可直接内存拷贝,`constexpr` 支持
- **三维运算**:加减乘除、点积、**三维叉积**、归一化
- **静态工厂函数**:`zero()`, `one()`, `unit_x/y/z()` 预定义常量
- **矩阵运算基础**:与 `Mat3`, `Mat4` 配合使用

## 类定义

```cpp
namespace mine::math {

struct Vec3 {
    float x{0.0f};
    float y{0.0f};
    float z{0.0f};

    // 构造函数
    constexpr Vec3() noexcept = default;
    constexpr Vec3(float x_value, float y_value, float z_value) noexcept;

    // 静态工厂函数
    [[nodiscard]] static constexpr Vec3 zero() noexcept;
    [[nodiscard]] static constexpr Vec3 one() noexcept;
    [[nodiscard]] static constexpr Vec3 unit_x() noexcept;
    [[nodiscard]] static constexpr Vec3 unit_y() noexcept;
    [[nodiscard]] static constexpr Vec3 unit_z() noexcept;

    // 索引访问
    [[nodiscard]] constexpr float& operator[](size_t index) noexcept;
    [[nodiscard]] constexpr const float& operator[](size_t index) const noexcept;

    // 一元运算符
    [[nodiscard]] constexpr Vec3 operator+() const noexcept;
    [[nodiscard]] constexpr Vec3 operator-() const noexcept;

    // 二元运算符
    [[nodiscard]] constexpr Vec3 operator+(Vec3 rhs) const noexcept;
    [[nodiscard]] constexpr Vec3 operator-(Vec3 rhs) const noexcept;
    [[nodiscard]] constexpr Vec3 operator*(float scalar) const noexcept;
    [[nodiscard]] constexpr Vec3 operator/(float scalar) const noexcept;

    // 复合赋值运算符
    constexpr Vec3& operator+=(Vec3 rhs) noexcept;
    constexpr Vec3& operator-=(Vec3 rhs) noexcept;
    constexpr Vec3& operator*=(float scalar) noexcept;
    constexpr Vec3& operator/=(float scalar) noexcept;

    // 向量运算
    [[nodiscard]] constexpr float dot(Vec3 rhs) const noexcept;
    [[nodiscard]] constexpr Vec3 cross(Vec3 rhs) const noexcept;  // 真正的三维叉积
    [[nodiscard]] constexpr float length_squared() const noexcept;
    [[nodiscard]] float length() const noexcept;
    [[nodiscard]] Vec3 normalized(float epsilon = kDefaultEpsilon) const noexcept;

    // 比较运算符
    [[nodiscard]] constexpr bool operator==(const Vec3&) const noexcept = default;
};

// 全局运算符
[[nodiscard]] constexpr Vec3 operator*(float scalar, Vec3 value) noexcept;

}
```

## 向量运算

### dot()

```cpp
[[nodiscard]] constexpr float dot(Vec3 rhs) const noexcept;
```

**描述**:计算点积（内积）。

**参数**:
- `rhs`:右操作数

**返回值**:`x * rhs.x + y * rhs.y + z * rhs.z`

**时间复杂度**:O(1)

**几何含义**:
- `|a| * |b| * cos(θ)`（其中 θ 是夹角）
- 点积为 0 表示垂直
- 点积为正表示夹角 < 90°
- 点积为负表示夹角 > 90°

**示例**:
```cpp
Vec3 a{1.0f, 0.0f, 0.0f};
Vec3 b{0.0f, 1.0f, 0.0f};
float d = a.dot(b);  // 0.0（垂直）

Vec3 c{1.0f, 1.0f, 0.0f};
float d2 = a.dot(c);  // 1.0（夹角 < 90°）
```

---

### cross()

```cpp
[[nodiscard]] constexpr Vec3 cross(Vec3 rhs) const noexcept;
```

**描述**:计算三维叉积（返回垂直于两个向量的新向量）。

**参数**:
- `rhs`:右操作数

**返回值**:
```cpp
Vec3{
    y * rhs.z - z * rhs.y,
    z * rhs.x - x * rhs.z,
    x * rhs.y - y * rhs.x
}
```

**时间复杂度**:O(1)

**几何含义**:
- 结果向量垂直于 `this` 和 `rhs`
- 方向遵循右手法则
- 长度等于平行四边形面积 `|a| * |b| * sin(θ)`
- 若共线则返回零向量

**示例**:
```cpp
Vec3 x_axis{1.0f, 0.0f, 0.0f};
Vec3 y_axis{0.0f, 1.0f, 0.0f};
Vec3 z_axis = x_axis.cross(y_axis);
// z_axis == Vec3{0.0f, 0.0f, 1.0f}

// 右手法则：X × Y = Z
// 反向：Y × X = -Z
Vec3 neg_z = y_axis.cross(x_axis);
// neg_z == Vec3{0.0f, 0.0f, -1.0f}
```

**常见用途**:
- **构造正交基**:从两个向量构造第三个垂直向量
- **计算法线**:给定三角形两边,叉积得到法线
- **判断方向**:判断点在平面哪一侧

```cpp
[[nodiscard]] constexpr float length_squared() const noexcept;
```

**描述**:计算向量长度的平方。

**返回值**:`x * x + y * y + z * z`

**时间复杂度**:O(1)

---

### length()

```cpp
[[nodiscard]] float length() const noexcept;
```

**描述**:计算向量长度（模）。

**返回值**:`std::sqrt(length_squared())`

**时间复杂度**:O(1),涉及平方根运算

---

### normalized()

```cpp
[[nodiscard]] Vec3 normalized(float epsilon = kDefaultEpsilon) const noexcept;
```

**描述**:返回归一化向量（长度为 1）。

**参数**:
- `epsilon`:长度阈值,低于此值返回零向量（默认 `1e-6f`）

**返回值**:
- 若 `length() > epsilon`：`*this / length()`
- 否则：`Vec3::zero()`

**时间复杂度**:O(1),涉及 `sqrt` 和除法

## 使用场景

### 1. 构造正交基

```cpp
// 给定一个前向向量,构造完整的坐标系
Vec3 forward{1.0f, 0.0f, 0.0f};
Vec3 world_up{0.0f, 1.0f, 0.0f};

Vec3 right = forward.cross(world_up).normalized();
Vec3 up = right.cross(forward);

// 现在有完整的右手坐标系：right, up, forward
```

---

### 2. 计算三角形法线

```cpp
Vec3 vertex0{0.0f, 0.0f, 0.0f};
Vec3 vertex1{1.0f, 0.0f, 0.0f};
Vec3 vertex2{0.0f, 1.0f, 0.0f};

Vec3 edge1 = vertex1 - vertex0;
Vec3 edge2 = vertex2 - vertex0;

Vec3 normal = edge1.cross(edge2).normalized();
// normal 指向三角形外侧
```

---

### 3. 判断点在平面哪一侧

```cpp
Vec3 plane_normal{0.0f, 1.0f, 0.0f};  // 向上的平面
Vec3 plane_point{0.0f, 0.0f, 0.0f};

Vec3 test_point{5.0f, 10.0f, 3.0f};
Vec3 to_point = test_point - plane_point;

float distance = plane_normal.dot(to_point);
if (distance > 0.0f) {
    // 在平面上方
} else if (distance < 0.0f) {
    // 在平面下方
} else {
    // 在平面上
}
```

---

### 4. 3D 物理模拟

```cpp
class RigidBody {
    Vec3 position_;
    Vec3 velocity_;
    Vec3 angular_velocity_;
    
public:
    void apply_force(Vec3 force, float dt) {
        Vec3 acceleration = force / mass_;
        velocity_ += acceleration * dt;
    }
    
    void apply_torque(Vec3 torque, float dt) {
        angular_velocity_ += torque * dt;
    }
    
    void update(float dt) {
        position_ += velocity_ * dt;
        // 旋转更新需要四元数或旋转矩阵
    }
};
```

### 5. 反射向量

```cpp
// 计算光线反射
Vec3 reflect(Vec3 incident, Vec3 normal) {
    // R = I - 2 * (I · N) * N
    return incident - normal * (2.0f * incident.dot(normal));
}

Vec3 light_dir{1.0f, -1.0f, 0.0f};
Vec3 surface_normal{0.0f, 1.0f, 0.0f};
Vec3 reflected = reflect(light_dir, surface_normal);
```

## 性能分析

### 内存布局

```cpp
sizeof(Vec3) == 12  // 3 * sizeof(float)
alignof(Vec3) == 4  // float 对齐
```

**注意**:SIMD 优化通常需要 16 字节对齐（Vec4）,`Vec3` 不直接支持。

---

### 运算性能

| 操作 | 时间复杂度 | 周期数（估算） | 说明 |
|------|-----------|-------------|------|
| 构造 | O(1) | ~1 | 寄存器赋值 |
| 加减 | O(1) | ~3 | 3 次浮点加法 |
| 标量乘除 | O(1) | ~3 | 3 次浮点乘法/除法 |
| `dot()` | O(1) | ~5 | 3 乘 + 2 加 |
| `cross()` | O(1) | ~9 | 6 乘 + 3 减 |
| `length_squared()` | O(1) | ~5 | 同 `dot()` |
| `length()` | O(1) | ~25 | `sqrt` 指令 |
| `normalized()` | O(1) | ~35 | `sqrt` + 3 除法 |

## 最佳实践

### 1. 使用叉积构造正交基

```cpp
// ✅ 推荐：Gram-Schmidt 正交化
Vec3 forward = (target - position).normalized();
Vec3 right = forward.cross(Vec3::unit_y()).normalized();
Vec3 up = right.cross(forward);
```

---

### 2. 法线计算注意顶点顺序

```cpp
// ✅ 正确：逆时针顶点顺序，法线向外
Vec3 edge1 = v1 - v0;
Vec3 edge2 = v2 - v0;
Vec3 normal = edge1.cross(edge2).normalized();

// ❌ 错误：顺时针顶点顺序，法线向内
Vec3 bad_normal = edge2.cross(edge1).normalized();
```

---

### 3. 避免冗余归一化

```cpp
// ❌ 不推荐：多次归一化
Vec3 a = v.normalized();
Vec3 b = a.normalized();  // 冗余

// ✅ 推荐：一次归一化
Vec3 dir = v.normalized();
```

## 常见陷阱

### 1. 共线向量叉积

```cpp
Vec3 a{1.0f, 0.0f, 0.0f};
Vec3 b{2.0f, 0.0f, 0.0f};  // 平行于 a
Vec3 c = a.cross(b);
// c == Vec3::zero()（退化）
```

---

### 2. SIMD 对齐问题

```cpp
// ❌ 不推荐：Vec3 不对齐 16 字节
std::vector<Vec3> vertices;  // 不利于 SIMD

// ✅ 推荐：使用 Vec4 或手动对齐
std::vector<Vec4> vertices_aligned;
```

## 总结

`Vec3` 是 `mine.math` 模块的基础 3D 向量类型,具备:

1. **POD 类型**:高性能,可 `constexpr`,ABI 稳定
2. **三维叉积**:`cross()` 返回垂直向量,遵循右手法则
3. **矩阵运算基础**:与 `Mat3`, `Mat4` 配合使用
4. **完整向量运算**:点积、叉积、归一化

**使用建议**:
- 叉积用于构造正交基和计算法线
- 注意顶点顺序影响法线方向
- 考虑 SIMD 优化时使用 Vec4
- 共线向量叉积返回零向量

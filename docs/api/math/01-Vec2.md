# Vec2 详细接口文档

## 概述

`Vec2` 是 `mine.math` 模块的 **二维浮点向量** 类型,表示 2D 空间中的向量、偏移或缩放系数。作为 POD 类型,`Vec2` 支持编译期常量、默认拷贝和移动,适用于高性能几何计算。

**核心特性:**
- **POD 类型**:Plain Old Data,可直接内存拷贝,`constexpr` 支持
- **浮点精度**:`float` 类型分量,平衡性能与精度
- **完整向量运算**:加减乘除、点积、叉积、归一化
- **静态工厂函数**:`zero()`, `one()`, `unit_x()`, `unit_y()` 预定义常量

## 类定义

```cpp
namespace mine::math {

struct Vec2 {
    float x{0.0f};
    float y{0.0f};

    // 构造函数
    constexpr Vec2() noexcept = default;
    constexpr Vec2(float x_value, float y_value) noexcept;

    // 静态工厂函数
    [[nodiscard]] static constexpr Vec2 zero() noexcept;
    [[nodiscard]] static constexpr Vec2 one() noexcept;
    [[nodiscard]] static constexpr Vec2 unit_x() noexcept;
    [[nodiscard]] static constexpr Vec2 unit_y() noexcept;

    // 索引访问
    [[nodiscard]] constexpr float& operator[](size_t index) noexcept;
    [[nodiscard]] constexpr const float& operator[](size_t index) const noexcept;

    // 一元运算符
    [[nodiscard]] constexpr Vec2 operator+() const noexcept;
    [[nodiscard]] constexpr Vec2 operator-() const noexcept;

    // 二元运算符
    [[nodiscard]] constexpr Vec2 operator+(Vec2 rhs) const noexcept;
    [[nodiscard]] constexpr Vec2 operator-(Vec2 rhs) const noexcept;
    [[nodiscard]] constexpr Vec2 operator*(float scalar) const noexcept;
    [[nodiscard]] constexpr Vec2 operator/(float scalar) const noexcept;

    // 复合赋值运算符
    constexpr Vec2& operator+=(Vec2 rhs) noexcept;
    constexpr Vec2& operator-=(Vec2 rhs) noexcept;
    constexpr Vec2& operator*=(float scalar) noexcept;
    constexpr Vec2& operator/=(float scalar) noexcept;

    // 向量运算
    [[nodiscard]] constexpr float dot(Vec2 rhs) const noexcept;
    [[nodiscard]] constexpr float cross(Vec2 rhs) const noexcept;
    [[nodiscard]] constexpr float length_squared() const noexcept;
    [[nodiscard]] float length() const noexcept;
    [[nodiscard]] Vec2 normalized(float epsilon = kDefaultEpsilon) const noexcept;

    // 比较运算符
    [[nodiscard]] constexpr bool operator==(const Vec2&) const noexcept = default;
};

// 全局运算符
[[nodiscard]] constexpr Vec2 operator*(float scalar, Vec2 value) noexcept;

}
```

## 二元运算符

### operator-

```cpp

```

**描述**:向量减法（分量逐个相减）。

**参数**:
- `rhs`:右操作数

**返回值**:`Vec2{x - rhs.x, y - rhs.y}`

**时间复杂度**:O(1)

**示例**:
```cpp

```

**示例**:

### operator/

```cpp
[[nodiscard]] constexpr Vec2 operator/(float scalar) const noexcept;
```

**注意**:调用者需确保 `scalar != 0.0f`。

**示例**:

## 向量运算

### dot()

```cpp
[[nodiscard]] constexpr float dot(Vec2 rhs) const noexcept;
```

**描述**:计算点积（内积）。

**参数**:
- `rhs`:右操作数

**返回值**:`x * rhs.x + y * rhs.y`

**时间复杂度**:O(1)

**几何含义**:
- `|a| * |b| * cos(θ)`（其中 θ 是夹角）
- 点积为 0 表示垂直
- 点积为正表示夹角 < 90°
- 点积为负表示夹角 > 90°

**示例**:
```cpp
Vec2 a{3.0f, 4.0f};
Vec2 b{1.0f, 0.0f};
float d = a.dot(b);  // 3.0

// 检查垂直
Vec2 right{1.0f, 0.0f};
Vec2 up{0.0f, 1.0f};
assert(right.dot(up) == 0.0f);  // 垂直
```

---

### cross()

```cpp
[[nodiscard]] constexpr float cross(Vec2 rhs) const noexcept;
```

**描述**:计算 2D 叉积（标量值,表示平行四边形有向面积）。

**参数**:
- `rhs`:右操作数

**返回值**:`x * rhs.y - y * rhs.x`

**时间复杂度**:O(1)

**几何含义**:
- 返回值 > 0：`rhs` 在 `this` 的逆时针方向
- 返回值 < 0：`rhs` 在 `this` 的顺时针方向
- 返回值 = 0：共线

**示例**:
```cpp
Vec2 a{1.0f, 0.0f};
Vec2 b{0.0f, 1.0f};
float c = a.cross(b);  // 1.0 (逆时针)

float c2 = b.cross(a);  // -1.0 (顺时针)

// 判断点在线段哪一侧
Vec2 line_dir{1.0f, 0.0f};
Vec2 point_offset{0.5f, 1.0f};
float side = line_dir.cross(point_offset);  // > 0 表示在左侧
```

---

### length_squared()

```cpp
[[nodiscard]] constexpr float length_squared() const noexcept;
```

**描述**:计算向量长度的平方。

**返回值**:`x * x + y * y`

**时间复杂度**:O(1)

**用途**:避免昂贵的平方根运算,用于距离比较。

**示例**:
```cpp
Vec2 v{3.0f, 4.0f};
float len_sq = v.length_squared();  // 25.0

// 比较距离（避免 sqrt）
Vec2 a{1.0f, 2.0f};
Vec2 b{4.0f, 6.0f};
if (a.length_squared() < b.length_squared()) {
    // a 更短
}
```

---

### length()

```cpp
[[nodiscard]] float length() const noexcept;
```

**描述**:计算向量长度（模）。

**返回值**:`std::sqrt(length_squared())`

**时间复杂度**:O(1),但涉及平方根运算（~20-30 周期）

**示例**:
```cpp
Vec2 v{3.0f, 4.0f};
float len = v.length();  // 5.0
```

---

### normalized()

```cpp
[[nodiscard]] Vec2 normalized(float epsilon = kDefaultEpsilon) const noexcept;
```

**描述**:返回归一化向量（长度为 1）。

**参数**:
- `epsilon`:长度阈值,低于此值返回零向量（默认 `1e-6f`）

**返回值**:
- 若 `length() > epsilon`：`*this / length()`
- 否则：`Vec2::zero()`

**时间复杂度**:O(1),涉及 `sqrt` 和除法

**示例**:
```cpp
Vec2 v{3.0f, 4.0f};
Vec2 dir = v.normalized();  // {0.6f, 0.8f}
assert(std::abs(dir.length() - 1.0f) < 1e-5f);

// 零向量安全处理
Vec2 zero{0.0f, 0.0f};
Vec2 result = zero.normalized();  // {0.0f, 0.0f}（不会崩溃）
```

## 比较运算符

### operator==

```cpp
[[nodiscard]] constexpr bool operator==(const Vec2&) const noexcept = default;
```

**描述**:逐分量精确比较。

**返回值**:`x == rhs.x && y == rhs.y`

**时间复杂度**:O(1)

**注意**:浮点精确比较,可能因舍入误差失败。推荐使用误差范围比较。

**示例**:
```cpp
Vec2 a{1.0f, 2.0f};
Vec2 b{1.0f, 2.0f};
assert(a == b);

// 浮点精度问题
Vec2 c = Vec2{1.0f, 2.0f} / 3.0f * 3.0f;
// c 可能不等于 {1.0f, 2.0f}（舍入误差）
```

## 使用场景

### 1. 2D 位置偏移

```cpp
Vec2 position{100.0f, 200.0f};
Vec2 velocity{10.0f, -5.0f};
float delta_time = 0.016f;

// 物理更新
position += velocity * delta_time;
```

---

### 2. 缩放系数

```cpp
Vec2 size{100.0f, 50.0f};
Vec2 scale{2.0f, 1.5f};

Vec2 scaled_size{size.x * scale.x, size.y * scale.y};
// 或使用逐分量乘法（需自定义）
```

---

### 3. 方向向量

```cpp
Vec2 from{0.0f, 0.0f};
Vec2 to{100.0f, 100.0f};
Vec2 direction = (to - from).normalized();

// 移动到目标
float speed = 50.0f;
Vec2 new_pos = from + direction * speed * delta_time;
```

---

### 4. 距离判断

```cpp
Vec2 a{0.0f, 0.0f};
Vec2 b{100.0f, 100.0f};
Vec2 c{200.0f, 200.0f};

// 避免 sqrt 比较距离
float dist_ab_sq = (b - a).length_squared();
float dist_ac_sq = (c - a).length_squared();

if (dist_ab_sq < dist_ac_sq) {
    // b 更近
}
```

---

### 5. 投影

```cpp
Vec2 a{3.0f, 4.0f};
Vec2 b{1.0f, 0.0f};

// 将 a 投影到 b
float proj_length = a.dot(b) / b.length();
Vec2 proj = b.normalized() * proj_length;
```

---

### 6. 角度判断

```cpp
Vec2 forward{1.0f, 0.0f};
Vec2 target{1.0f, 1.0f};

// 叉积判断左右
float cross = forward.cross(target);
if (cross > 0.0f) {
    // target 在 forward 左侧（逆时针）
} else if (cross < 0.0f) {
    // target 在 forward 右侧（顺时针）
} else {
    // 共线
}
```

## 性能分析

### 内存布局

```cpp
sizeof(Vec2) == 8  // 2 * sizeof(float)
alignof(Vec2) == 4  // float 对齐
```

**POD 类型**:
- 可 `memcpy` 复制
- 可用于 `std::vector<Vec2>`（连续存储）
- 可跨 DLL 边界传递

---

### 运算性能

| 操作 | 时间复杂度 | 周期数（估算） | 说明 |
|------|-----------|-------------|------|
| 构造 | O(1) | ~1 | 寄存器赋值 |
| 加减 | O(1) | ~2 | 2 次浮点加法 |
| 标量乘除 | O(1) | ~2 | 2 次浮点乘法/除法 |
| `dot()` | O(1) | ~3 | 2 乘 + 1 加 |
| `cross()` | O(1) | ~3 | 2 乘 + 1 减 |
| `length_squared()` | O(1) | ~3 | 同 `dot()` |
| `length()` | O(1) | ~25 | `sqrt` 指令 |
| `normalized()` | O(1) | ~30 | `sqrt` + 除法 |

---

### 编译期计算

所有 `constexpr` 函数可在编译期求值:

```cpp
constexpr Vec2 v1{3.0f, 4.0f};
constexpr Vec2 v2{1.0f, 2.0f};
constexpr Vec2 sum = v1 + v2;
constexpr float d = v1.dot(v2);

static_assert(sum.x == 4.0f && sum.y == 6.0f);
static_assert(d == 11.0f);
```

**优势**:
- 无运行时开销
- 编译器优化为常量

## 最佳实践

### 1. 使用静态工厂函数

```cpp
// ✅ 推荐：语义清晰
Vec2 zero = Vec2::zero();
Vec2 right = Vec2::unit_x();

// ❌ 不推荐：不够清晰
Vec2 zero2{0.0f, 0.0f};
Vec2 right2{1.0f, 0.0f};
```

---

### 2. 浮点比较使用误差范围

```cpp
// ❌ 不推荐：精确比较
bool equal = (a == b);

// ✅ 推荐：误差范围比较
bool approx_equal(Vec2 a, Vec2 b, float epsilon = 1e-5f) {
    Vec2 diff = a - b;
    return diff.length_squared() < epsilon * epsilon;
}
```

---

### 3. 避免不必要的 sqrt

```cpp
// ❌ 不推荐：多次 sqrt
if (v.length() < threshold) { ... }
if ((a - b).length() < dist) { ... }

// ✅ 推荐：平方比较
if (v.length_squared() < threshold * threshold) { ... }
if ((a - b).length_squared() < dist * dist) { ... }
```

---

### 4. 使用 `constexpr` 定义常量

```cpp
// ✅ 推荐：编译期常量
constexpr Vec2 kGravity{0.0f, -9.8f};
constexpr Vec2 kOrigin = Vec2::zero();
```

---

### 5. 归一化前检查长度

```cpp
// ✅ 推荐：normalized() 内置零检查
Vec2 dir = v.normalized();  // 安全

// ❌ 不推荐：手动检查（冗余）
Vec2 dir2;
if (v.length() > 0.0f) {
    dir2 = v / v.length();  // 两次 sqrt!
}
```

## 常见陷阱

### 1. 零向量归一化

```cpp
// ❌ 不推荐：手动除法可能除零
Vec2 v{0.0f, 0.0f};
Vec2 dir = v / v.length();  // NaN!

// ✅ 推荐：使用 normalized()
Vec2 dir2 = v.normalized();  // 安全返回零向量
```

---

### 2. 浮点精度丢失

```cpp
Vec2 v{1e8f, 1.0f};
v += Vec2{1e-6f, 0.0f};  // 1e-6 被吞没（精度不足）
```

---

### 3. 跨单位系统混用

```cpp
// ❌ 不推荐：混用像素和米
Vec2 pos_pixels{100.0f, 200.0f};
Vec2 velocity_meters{10.0f, 5.0f};
pos_pixels += velocity_meters;  // 单位不匹配!

// ✅ 推荐：统一单位或类型安全包装
Vec2 velocity_pixels = velocity_meters * pixels_per_meter;
pos_pixels += velocity_pixels;
```

## 完整示例

### 示例 1：简单物理模拟

```cpp
#include <mine/math/Vec2.h>

using namespace mine::math;

class Particle {
    Vec2 position_;
    Vec2 velocity_;
    
public:
    Particle(Vec2 pos) : position_{pos}, velocity_{Vec2::zero()} {}
    
    void apply_force(Vec2 force, float dt) {
        velocity_ += force * dt;
    }
    
    void update(float dt) {
        position_ += velocity_ * dt;
    }
    
    Vec2 position() const { return position_; }
};

void example() {
    Particle p{Vec2{0.0f, 100.0f}};
    Vec2 gravity{0.0f, -9.8f};
    float dt = 0.016f;
    
    for (int i = 0; i < 100; ++i) {
        p.apply_force(gravity, dt);
        p.update(dt);
        std::cout << "Frame " << i << ": " << p.position().y << "\n";
    }
}
```

### 示例 2：碰撞检测

```cpp
bool circle_intersects(Vec2 center_a, float radius_a,
                       Vec2 center_b, float radius_b) {
    Vec2 diff = center_b - center_a;
    float dist_sq = diff.length_squared();
    float radius_sum = radius_a + radius_b;
    return dist_sq < radius_sum * radius_sum;
}
```

### 示例 3：2D 转向行为

```cpp
Vec2 seek(Vec2 position, Vec2 target, Vec2 velocity, float max_speed) {
    Vec2 desired = (target - position).normalized() * max_speed;
    Vec2 steering = desired - velocity;
    return steering;
}

void example() {
    Vec2 pos{0.0f, 0.0f};
    Vec2 target{100.0f, 100.0f};
    Vec2 velocity{10.0f, 0.0f};
    float max_speed = 50.0f;
    
    Vec2 steering = seek(pos, target, velocity, max_speed);
    velocity += steering * 0.1f;  // 应用转向力
}
```

## 总结

`Vec2` 是 `mine.math` 模块的基础 2D 向量类型,具备:

1. **POD 类型**:高性能,可 `constexpr`,ABI 稳定
2. **完整运算**:加减乘除、点积、叉积、归一化
3. **静态常量**:`zero()`, `one()`, `unit_x()`, `unit_y()`
4. **安全归一化**:`normalized()` 内置零检查

**使用建议**:
- 优先使用 `length_squared()` 避免 `sqrt`
- 浮点比较使用误差范围
- `constexpr` 定义编译期常量
- 单位系统保持一致

# Vec4 详细接口文档

## 概述

`Vec4` 是 `mine.math` 模块的 **四维浮点向量** 类型,主要用于齐次坐标、颜色运算和 SIMD 优化。作为 POD 类型,`Vec4` 的 16 字节对齐特性使其非常适合现代 CPU 的向量指令集。

**核心特性:**
- **POD 类型**:Plain Old Data,可直接内存拷贝,`constexpr` 支持
- **16 字节对齐**:适配 SSE/NEON 等 SIMD 指令集
- **齐次坐标**:第 4 分量 `w` 用于透视变换和矩阵乘法
- **颜色运算**:RGBA 颜色存储和计算（虽然 `Color` 类更常用）

**与 Vec3 的区别:**
- **无叉积**:`Vec4` 不支持叉积运算（叉积是三维概念）
- **齐次坐标**:`w` 分量用于齐次坐标系（点为 1,向量为 0）
- **SIMD 友好**:16 字节对齐,单条指令处理 4 个浮点

## 类定义

```cpp
namespace mine::math {

struct Vec4 {
    float x{0.0f};
    float y{0.0f};
    float z{0.0f};
    float w{0.0f};

    // 构造函数
    constexpr Vec4() noexcept = default;
    constexpr Vec4(float x_value, float y_value, float z_value, float w_value) noexcept;

    // 静态工厂函数
    [[nodiscard]] static constexpr Vec4 zero() noexcept;
    [[nodiscard]] static constexpr Vec4 one() noexcept;

    // 索引访问
    [[nodiscard]] constexpr float& operator[](size_t index) noexcept;
    [[nodiscard]] constexpr const float& operator[](size_t index) const noexcept;

    // 一元运算符
    [[nodiscard]] constexpr Vec4 operator+() const noexcept;
    [[nodiscard]] constexpr Vec4 operator-() const noexcept;

    // 二元运算符
    [[nodiscard]] constexpr Vec4 operator+(Vec4 rhs) const noexcept;
    [[nodiscard]] constexpr Vec4 operator-(Vec4 rhs) const noexcept;
    [[nodiscard]] constexpr Vec4 operator*(float scalar) const noexcept;
    [[nodiscard]] constexpr Vec4 operator/(float scalar) const noexcept;

    // 复合赋值运算符
    constexpr Vec4& operator+=(Vec4 rhs) noexcept;
    constexpr Vec4& operator-=(Vec4 rhs) noexcept;
    constexpr Vec4& operator*=(float scalar) noexcept;
    constexpr Vec4& operator/=(float scalar) noexcept;

    // 向量运算
    [[nodiscard]] constexpr float dot(Vec4 rhs) const noexcept;
    [[nodiscard]] constexpr float length_squared() const noexcept;
    [[nodiscard]] float length() const noexcept;
    [[nodiscard]] Vec4 normalized(float epsilon = kDefaultEpsilon) const noexcept;

    // 比较运算符
    [[nodiscard]] constexpr bool operator==(const Vec4&) const noexcept = default;
};

// 全局运算符
[[nodiscard]] constexpr Vec4 operator*(float scalar, Vec4 value) noexcept;

}
```

## 成员变量

### x / y / z / w

```cpp
float x{0.0f};
float y{0.0f};
float z{0.0f};
float w{0.0f};
```

**描述**:向量的四个分量,公开访问。

**默认值**:`0.0f`（零向量）

**齐次坐标语义**:
- **点**:`w = 1.0f`（受平移影响）
- **向量**:`w = 0.0f`（不受平移影响）
- **颜色**:`x, y, z, w` 对应 `R, G, B, A`

**示例**:
```cpp
Vec4 point{10.0f, 20.0f, 30.0f, 1.0f};  // 齐次坐标的点
Vec4 direction{0.0f, 1.0f, 0.0f, 0.0f};  // 齐次坐标的方向
Vec4 color{1.0f, 0.5f, 0.25f, 1.0f};     // RGBA 颜色
```

## 构造函数

### 默认构造函数

```cpp

```

**描述**:创建零向量 `(0, 0, 0, 0)`。

**时间复杂度**:O(1)

**示例**:
```cpp
constexpr Vec4 v;
static_assert(v.x == 0.0f && v.y == 0.0f && v.z == 0.0f && v.w == 0.0f);
```

---

```cpp

```

**参数**:
- `x_value`:X 分量
- `y_value`:Y 分量
- `z_value`:Z 分量
- `w_value`:W 分量

**时间复杂度**:O(1)

**示例**:
```cpp
constexpr Vec4 v{1.0f, 2.0f, 3.0f, 4.0f};
static_assert(v.w == 4.0f);
```


## 二元运算符

### operator+

```cpp
[[nodiscard]] constexpr Vec4 operator+(Vec4 rhs) const noexcept;
```

**描述**:向量加法（分量逐个相加）。

**参数**:
- `rhs`:右操作数

**返回值**:`Vec4{x + rhs.x, y + rhs.y, z + rhs.z, w + rhs.w}`

**时间复杂度**:O(1)（SIMD 优化可单条指令完成）

**示例**:
```cpp
constexpr Vec4 a{1.0f, 2.0f, 3.0f, 4.0f};
constexpr Vec4 b{5.0f, 6.0f, 7.0f, 8.0f};
constexpr Vec4 c = a + b;
static_assert(c.x == 6.0f && c.w == 12.0f);
```

---

### operator-

```cpp
[[nodiscard]] constexpr Vec4 operator-(Vec4 rhs) const noexcept;
```

**描述**:向量减法（分量逐个相减）。

**参数**:
- `rhs`:右操作数

**返回值**:`Vec4{x - rhs.x, y - rhs.y, z - rhs.z, w - rhs.w}`

**时间复杂度**:O(1)

---

### operator*

```cpp
[[nodiscard]] constexpr Vec4 operator*(float scalar) const noexcept;
```

**描述**:标量乘法（缩放向量）。

**参数**:
- `scalar`:缩放因子

**返回值**:`Vec4{x * scalar, y * scalar, z * scalar, w * scalar}`

**时间复杂度**:O(1)（SIMD 可单条指令）

---

### operator/

```cpp
[[nodiscard]] constexpr Vec4 operator/(float scalar) const noexcept;
```

**描述**:标量除法。

**参数**:
- `scalar`:除数（**不检查零除**）

**返回值**:`Vec4{x / scalar, y / scalar, z / scalar, w / scalar}`

**时间复杂度**:O(1)

## 向量运算

### dot()

```cpp
[[nodiscard]] constexpr float dot(Vec4 rhs) const noexcept;
```

**描述**:计算点积（内积）。

**参数**:
- `rhs`:右操作数

**返回值**:`x * rhs.x + y * rhs.y + z * rhs.z + w * rhs.w`

**时间复杂度**:O(1)（SIMD 可单条指令）

**示例**:
```cpp
Vec4 a{1.0f, 2.0f, 3.0f, 4.0f};
Vec4 b{5.0f, 6.0f, 7.0f, 8.0f};
float d = a.dot(b);  // 1*5 + 2*6 + 3*7 + 4*8 = 70.0
```

---

### length_squared()

```cpp
[[nodiscard]] constexpr float length_squared() const noexcept;
```

**描述**:计算向量长度的平方。

**返回值**:`x * x + y * y + z * z + w * w`

**时间复杂度**:O(1)

---

### length()

```cpp
[[nodiscard]] float length() const noexcept;
```

**描述**:计算向量长度（模）。

**返回值**:`std::sqrt(length_squared())`

**时间复杂度**:O(1),涉及平方根运算

---

### normalized()

```cpp
[[nodiscard]] Vec4 normalized(float epsilon = kDefaultEpsilon) const noexcept;
```

**描述**:返回归一化向量（长度为 1）。

**参数**:
- `epsilon`:长度阈值,低于此值返回零向量（默认 `1e-6f`）

**返回值**:
- 若 `length() > epsilon`：`*this / length()`
- 否则：`Vec4::zero()`

**时间复杂度**:O(1),涉及 `sqrt` 和 4 次除法

## 使用场景

### 1. 齐次坐标变换

```cpp
// 点在齐次坐标系中
Vec4 point{10.0f, 20.0f, 30.0f, 1.0f};

// 矩阵变换（包括平移）
Mat4 transform = Mat4::translation(5.0f, 10.0f, 15.0f);
Vec4 transformed = transform * point;
// transformed.w == 1.0f（点保持为点）

// 方向向量（不受平移影响）
Vec4 direction{0.0f, 1.0f, 0.0f, 0.0f};
Vec4 rotated = transform * direction;
// rotated.w == 0.0f（方向保持为方向）
```

---

### 2. SIMD 批量处理

```cpp
// 批量顶点变换（利用 SIMD）
void transform_vertices(Mat4 mvp, Span<Vec4> vertices) {
    for (Vec4& v : vertices) {
        v = mvp * v;  // SIMD 优化的矩阵乘法
    }
}
```

---

### 3. 颜色插值

```cpp
Vec4 color_start{1.0f, 0.0f, 0.0f, 1.0f};  // 红色
Vec4 color_end{0.0f, 0.0f, 1.0f, 1.0f};     // 蓝色

// 线性插值
float t = 0.5f;
Vec4 lerp = color_start * (1.0f - t) + color_end * t;
// lerp == {0.5f, 0.0f, 0.5f, 1.0f}（紫色）
```

---

### 4. 透视除法

```cpp
// 裁剪空间 → NDC（归一化设备坐标）
Vec4 clip_pos{100.0f, 50.0f, 200.0f, 400.0f};

// 透视除法
Vec3 ndc{
    clip_pos.x / clip_pos.w,  // 0.25
    clip_pos.y / clip_pos.w,  // 0.125
    clip_pos.z / clip_pos.w   // 0.5
};
```

---

### 5. 平面方程

```cpp
// 平面方程：Ax + By + Cz + D = 0
// 存储为 Vec4{A, B, C, D}
Vec4 plane{0.0f, 1.0f, 0.0f, -10.0f};  // y - 10 = 0（y = 10 的水平面）

// 点到平面的距离
Vec4 point{5.0f, 20.0f, 3.0f, 1.0f};
float distance = plane.x * point.x + 
                 plane.y * point.y + 
                 plane.z * point.z + 
                 plane.w;
// distance == 10.0（点在平面上方 10 单位）
```

## 性能分析

### 内存布局

```cpp
sizeof(Vec4) == 16  // 4 * sizeof(float)
alignof(Vec4) == 4  // float 对齐（注意：SIMD 可能需要 16 字节对齐）
```

**SIMD 友好**:
- **SSE**:128 位寄存器,一次处理 4 个 float
- **AVX**:256 位寄存器,一次处理 8 个 float（2 个 Vec4）
- **NEON**:128 位寄存器,一次处理 4 个 float

---

### 运算性能（SIMD 优化）

| 操作 | 标量周期数 | SIMD 周期数 | 加速比 |
|------|----------|-----------|-------|
| 加减 | ~4 | ~1 | 4x |
| 标量乘 | ~4 | ~1 | 4x |
| 点积 | ~7 | ~3 | 2.3x |
| 归一化 | ~35 | ~10 | 3.5x |

**注意**:SIMD 加速需要编译器优化和数据对齐。

---

### 编译期计算

所有 `constexpr` 函数可在编译期求值:

```cpp
constexpr Vec4 v1{1.0f, 2.0f, 3.0f, 4.0f};
constexpr Vec4 v2{5.0f, 6.0f, 7.0f, 8.0f};
constexpr Vec4 sum = v1 + v2;
constexpr float d = v1.dot(v2);

static_assert(sum.w == 12.0f);
static_assert(d == 70.0f);
```

## 常见陷阱

### 1. 忘记齐次坐标的 w 分量

```cpp
// ❌ 错误：默认构造的 w = 0.0（方向而非点）
Vec4 pos{10.0f, 20.0f, 30.0f, 0.0f};
Vec4 translated = translation_matrix * pos;
// translated == pos（平移无效!）

// ✅ 正确：点的 w = 1.0
Vec4 pos_correct{10.0f, 20.0f, 30.0f, 1.0f};
```

---

### 2. Vec4 不支持叉积

```cpp
// ❌ 错误：Vec4 没有 cross() 方法
Vec4 a{1.0f, 0.0f, 0.0f, 0.0f};
Vec4 b{0.0f, 1.0f, 0.0f, 0.0f};
// Vec4 c = a.cross(b);  // 编译错误!

// ✅ 正确：使用 Vec3
Vec3 a3{1.0f, 0.0f, 0.0f};
Vec3 b3{0.0f, 1.0f, 0.0f};
Vec3 c3 = a3.cross(b3);
```

---

### 3. 透视除法丢失深度信息

```cpp
// ❌ 错误：过早透视除法丢失 w（深度）
Vec4 clip = mvp * point;
Vec3 ndc{clip.x / clip.w, clip.y / clip.w, clip.z / clip.w};
// 后续无法恢复 clip.w 进行深度比较

// ✅ 正确：保留 clip 空间坐标
Vec4 clip_space = mvp * point;
// 深度测试后再透视除法
```

## 完整示例

### 示例：顶点着色器模拟

```cpp
#include <mine/math/Vec4.h>
#include <mine/math/Mat4.h>

struct Vertex {
    Vec3 position;
    Vec3 normal;
    Vec2 uv;
};

struct VertexShaderOutput {
    Vec4 clip_position;
    Vec3 world_normal;
    Vec2 uv;
};

VertexShaderOutput vertex_shader(const Vertex& input,
                                  const Mat4& model,
                                  const Mat4& view,
                                  const Mat4& projection) {
    // 模型空间 → 世界空间
    Vec4 world_pos = model * Vec4{input.position.x, input.position.y, 
                                   input.position.z, 1.0f};
    
    // 世界空间 → 裁剪空间
    Vec4 clip_pos = projection * (view * world_pos);
    
    // 法线变换（只旋转，不平移）
    Vec4 world_normal = model * Vec4{input.normal.x, input.normal.y, 
                                      input.normal.z, 0.0f};
    
    return {
        .clip_position = clip_pos,
        .world_normal = Vec3{world_normal.x, world_normal.y, world_normal.z},
        .uv = input.uv
    };
}
```

## 总结

`Vec4` 是 `mine.math` 模块的四维向量类型,具备:

1. **POD 类型**:高性能,可 `constexpr`,16 字节大小
2. **齐次坐标**:用于矩阵变换（点 w=1,方向 w=0）
3. **SIMD 友好**:16 字节对齐,适配 SSE/NEON
4. **无叉积**:叉积是三维概念,Vec4 不支持

**使用建议**:
- 齐次坐标正确设置 w 分量
- SIMD 批量处理时注意数据对齐
- 颜色存储优先使用 `Color` 类
- 需要叉积时使用 `Vec3`

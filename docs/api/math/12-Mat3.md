# Mat3 详细接口文档

## 概述

`Mat3` 是 `mine.math` 模块的 **3x3 浮点矩阵** 类型，主要用于二维仿射变换（平移、旋转、缩放）。采用**行主序**存储，按**列向量**语义参与运算。

**核心特性：**
- **行主序存储**：内存布局 `m[row][col]`
- **列向量语义**：矩阵乘以列向量（`M * v`）
- **2D 仿射变换**：支持平移、旋转、缩放及组合
- **齐次坐标**：第三行 `[0, 0, 1]` 表示仿射变换

## 类定义

```cpp
namespace mine::math {

struct Mat3 {
    float m[3][3];  // 默认初始化为单位矩阵

    // 构造函数
    constexpr Mat3() noexcept = default;  // 单位矩阵
    constexpr Mat3(float m00, float m01, float m02,
                   float m10, float m11, float m12,
                   float m20, float m21, float m22) noexcept;

    // 静态工厂函数
    [[nodiscard]] static constexpr Mat3 zero() noexcept;
    [[nodiscard]] static constexpr Mat3 identity() noexcept;
    [[nodiscard]] static constexpr Mat3 translation(float tx, float ty) noexcept;
    [[nodiscard]] static constexpr Mat3 scaling(float sx, float sy) noexcept;
    [[nodiscard]] static Mat3 rotation(float radians_value) noexcept;

    // 元素访问
    [[nodiscard]] constexpr float* operator[](size_t row) noexcept;
    [[nodiscard]] constexpr const float* operator[](size_t row) const noexcept;
    [[nodiscard]] constexpr Vec3 row(size_t index) const noexcept;
    [[nodiscard]] constexpr Vec3 column(size_t index) const noexcept;

    // 矩阵运算
    [[nodiscard]] constexpr Mat3 operator*(const Mat3& rhs) const noexcept;
    constexpr Mat3& operator*=(const Mat3& rhs) noexcept;
    [[nodiscard]] constexpr Vec3 operator*(Vec3 rhs) const noexcept;

    // 2D 变换
    [[nodiscard]] constexpr Vec2 transform_point(Vec2 point) const noexcept;
    [[nodiscard]] constexpr Vec2 transform_vector(Vec2 vector) const noexcept;

    // 矩阵操作
    [[nodiscard]] constexpr Mat3 transposed() const noexcept;
    [[nodiscard]] constexpr float determinant() const noexcept;
    [[nodiscard]] mine::core::Result<Mat3> inverted(float epsilon = kDefaultEpsilon) const noexcept;

    // 比较运算符
    [[nodiscard]] constexpr bool operator==(const Mat3&) const noexcept = default;
};

}
```

**参数**：

**返回值**：

### scaling()

```cpp
[[nodiscard]] static constexpr Mat3 scaling(float sx, float sy) noexcept;
```

**描述**：创建缩放矩阵。

**参数**：
- `sx`, `sy`：x, y 方向的缩放因子

**返回值**：
```cpp
{sx, 0,  0}
{0,  sy, 0}
{0,  0,  1}
```

**示例**：
```cpp
Mat3 s = Mat3::scaling(2, 3);
Vec2 v{10, 10};
Vec2 scaled = s.transform_vector(v);  // {20, 30}
```

---

### rotation()

```cpp
[[nodiscard]] static Mat3 rotation(float radians_value) noexcept;
```

**描述**：创建旋转矩阵（逆时针，绕原点）。

**参数**：
- `radians_value`：旋转角度（弧度）

**返回值**：
```cpp
{cos(θ), -sin(θ), 0}
{sin(θ),  cos(θ), 0}
{0,       0,      1}
```

**示例**：
```cpp
float angle = std::numbers::pi_v<float> / 2;  // 90度
Mat3 r = Mat3::rotation(angle);
Vec2 v{1, 0};
Vec2 rotated = r.transform_vector(v);  // {0, 1}
```

## 矩阵运算

### operator*（矩阵乘法）

```cpp
[[nodiscard]] constexpr Mat3 operator*(const Mat3& rhs) const noexcept;
```

**描述**：矩阵乘法（`this * rhs`），行主序实现。

**返回值**：`result[i][j] = row(i).dot(rhs.column(j))`

**时间复杂度**：O(27)（3x3x3 乘法）

**示例**：
```cpp
Mat3 t = Mat3::translation(10, 0);
Mat3 r = Mat3::rotation(std::numbers::pi_v<float> / 2);
Mat3 combined = t * r;  // 先旋转，再平移
```

**注意**：矩阵乘法不满足交换律（`A * B != B * A`）

---

### operator*（矩阵-向量乘法）

```cpp
[[nodiscard]] constexpr Vec3 operator*(Vec3 rhs) const noexcept;
```

**描述**：矩阵乘以列向量（`M * v`）。

**返回值**：
```cpp
{row(0).dot(rhs),
 row(1).dot(rhs),
 row(2).dot(rhs)}
```

**时间复杂度**：O(9)（3x3 乘法）

**示例**：
```cpp
Mat3 m = Mat3::translation(5, 10);
Vec3 v{1, 2, 1};  // 齐次坐标点 (1, 2)
Vec3 result = m * v;  // {6, 12, 1} → 点 (6, 12)
```

## 2D 变换

### transform_point()

```cpp
[[nodiscard]] constexpr Vec2 transform_point(Vec2 point) const noexcept;
```

**描述**：变换 2D 点（受平移影响）。

**实现**：`M * Vec3{point.x, point.y, 1.0}`，取前两个分量

**示例**：
```cpp
Mat3 t = Mat3::translation(10, 20);
Vec2 p{5, 5};
Vec2 transformed = t.transform_point(p);  // {15, 25}
```

---

### transform_vector()

```cpp
[[nodiscard]] constexpr Vec2 transform_vector(Vec2 vector) const noexcept;
```

**描述**：变换 2D 向量（**不受平移影响**）。

**实现**：`M * Vec3{vector.x, vector.y, 0.0}`，取前两个分量

**示例**：
```cpp
Mat3 t = Mat3::translation(10, 20);
Vec2 v{1, 0};
Vec2 transformed = t.transform_vector(v);  // {1, 0}（平移不影响向量）

Mat3 s = Mat3::scaling(2, 3);
Vec2 scaled = s.transform_vector(v);  // {2, 0}
```

## 矩阵操作

### transposed()

```cpp
[[nodiscard]] constexpr Mat3 transposed() const noexcept;
```

**描述**：返回转置矩阵（行列互换）。

**返回值**：`result[i][j] = m[j][i]`

**时间复杂度**：O(9)

---

### determinant()

```cpp
[[nodiscard]] constexpr float determinant() const noexcept;
```

**描述**：计算行列式。

**公式**（萨吕斯法则）：
```
det = m[0][0] * (m[1][1] * m[2][2] - m[1][2] * m[2][1])
    - m[0][1] * (m[1][0] * m[2][2] - m[1][2] * m[2][0])
    + m[0][2] * (m[1][0] * m[2][1] - m[1][1] * m[2][0])
```

**时间复杂度**：O(17)

**用途**：
- `det == 0`：矩阵不可逆
- `det < 0`：变换包含镜像翻转

**示例**：
```cpp
Mat3 m = Mat3::identity();
assert(m.determinant() == 1.0f);

Mat3 s = Mat3::scaling(2, 3);
assert(s.determinant() == 6.0f);  // sx * sy
```

---

### inverted()

```cpp
[[nodiscard]] mine::core::Result<Mat3> inverted(float epsilon = kDefaultEpsilon) const noexcept;
```

**描述**：计算逆矩阵。

**返回值**：
- 成功：`Result<Mat3>` 包含逆矩阵
- 失败（`|det| <= epsilon`）：`Result<Mat3>` 包含错误状态

**时间复杂度**：O(50)（伴随矩阵法）

**示例**：
```cpp
Mat3 t = Mat3::translation(10, 20);
auto inv_result = t.inverted();
if (inv_result) {
    Mat3 inv = inv_result.value();
    Vec2 p{15, 25};
    Vec2 original = inv.transform_point(p);  // {5, 5}
}
```

## 使用场景

### 1. 组合变换

```cpp
// 先缩放，再旋转，最后平移
Mat3 scale = Mat3::scaling(2, 2);
Mat3 rotate = Mat3::rotation(std::numbers::pi_v<float> / 4);  // 45度
Mat3 translate = Mat3::translation(100, 50);

Mat3 combined = translate * rotate * scale;
// 注意顺序：最右边的变换最先应用

Vec2 point{10, 0};
Vec2 transformed = combined.transform_point(point);
```

---

### 2. UI 控件变换

```cpp
class TransformedWidget {
    Mat3 transform_ = Mat3::identity();
    
public:
    void set_position(Vec2 pos) {
        transform_ = Mat3::translation(pos.x, pos.y) * 
                     Mat3::rotation(rotation_) * 
                     Mat3::scaling(scale_.x, scale_.y);
    }
    
    bool hit_test(Vec2 screen_pos) {
        auto inv = transform_.inverted();
        if (!inv) return false;
        
        Vec2 local_pos = inv.value().transform_point(screen_pos);
        return local_bounds_.contains(local_pos);
    }
};
```

---

### 3. 相机变换

```cpp
Mat3 calculate_view_matrix(Vec2 camera_pos, float zoom) {
    // 相机变换 = 先缩放（zoom），再平移（-camera_pos）
    Mat3 translate = Mat3::translation(-camera_pos.x, -camera_pos.y);
    Mat3 scale = Mat3::scaling(zoom, zoom);
    return scale * translate;
}

// 使用
Vec2 world_pos{100, 100};
Mat3 view = calculate_view_matrix({50, 50}, 2.0f);
Vec2 screen_pos = view.transform_point(world_pos);  // {100, 100}
```

## 最佳实践

### 1. 变换顺序（从右到左）

```cpp
// ✅ 推荐：清晰的变换顺序
Mat3 m = translate * rotate * scale;
// 应用时：先 scale，再 rotate，最后 translate

// ❌ 容易混淆的顺序
Mat3 m2 = scale * rotate * translate;
// 实际效果与预期不符
```

---

### 2. 区分点和向量

```cpp
// ✅ 推荐：使用专用方法
Vec2 transformed_point = m.transform_point(point);
Vec2 transformed_vector = m.transform_vector(vector);

// ❌ 不推荐：手动构造 Vec3
Vec2 p2 = (m * Vec3{point.x, point.y, 1.0f}).xy();  // 冗长
```

---

### 3. 检查逆矩阵是否存在

```cpp
// ✅ 推荐：检查 Result
auto inv_result = m.inverted();
if (inv_result) {
    Mat3 inv = inv_result.value();
    // 使用逆矩阵
} else {
    // 处理不可逆情况
}

// ❌ 不推荐：不检查直接使用
Mat3 inv = m.inverted().value();  // 可能抛出异常
```

## 常见陷阱

### 1. 矩阵乘法不满足交换律

```cpp
Mat3 t = Mat3::translation(10, 0);
Mat3 r = Mat3::rotation(std::numbers::pi_v<float> / 2);

Mat3 a = t * r;  // 先旋转，再平移
Mat3 b = r * t;  // 先平移，再旋转
// a != b
```

---

### 2. transform_vector() 不受平移影响

```cpp
Mat3 t = Mat3::translation(100, 100);
Vec2 v{1, 0};
Vec2 transformed = t.transform_vector(v);
// transformed == {1, 0}（向量不受平移影响！）
```

## 完整示例

```cpp
#include <mine/math/Mat3.h>
#include <mine/math/Vec2.h>

// 示例：旋转图片围绕中心点
Mat3 rotate_around_center(Size image_size, float radians) {
    Vec2 center{image_size.width * 0.5f, image_size.height * 0.5f};
    
    // 1. 平移到原点
    Mat3 to_origin = Mat3::translation(-center.x, -center.y);
    
    // 2. 旋转
    Mat3 rotate = Mat3::rotation(radians);
    
    // 3. 平移回中心
    Mat3 to_center = Mat3::translation(center.x, center.y);
    
    // 组合：先到原点，再旋转，最后回到中心
    return to_center * rotate * to_origin;
}

// 使用
void example() {
    Size image{200, 100};
    float angle = std::numbers::pi_v<float> / 4;  // 45度
    Mat3 transform = rotate_around_center(image, angle);
    
    // 变换图片四个角
    Vec2 tl = transform.transform_point({0, 0});
    Vec2 tr = transform.transform_point({200, 0});
    Vec2 bl = transform.transform_point({0, 100});
    Vec2 br = transform.transform_point({200, 100});
    
    // 绘制旋转后的图片
    draw_transformed_image(image, {tl, tr, br, bl});
}
```

## 总结

`Mat3` 是 `mine.math` 模块的 3x3 矩阵类型，具备：

1. **2D 仿射变换**：平移、旋转、缩放及组合
2. **行主序存储，列向量语义**：内存连续，适合缓存
3. **完整矩阵运算**：乘法、转置、行列式、逆矩阵
4. **点/向量区分**：`transform_point()` vs `transform_vector()`

**使用建议**：
- 变换顺序从右到左（`translate * rotate * scale`）
- 使用 `transform_point/vector()` 而非手动构造 Vec3
- 检查 `inverted()` 返回的 Result
- 注意矩阵乘法不满足交换律

# Utf8 详细接口文档

## 概述

`Utf8.h` 提供了 **UTF-8 编解码与光标移动工具函数**，用于独立于平台的 UTF-8 字符边界遍历、UTF-32↔UTF-8 转换。

**核心特性：**
- **4 个工具函数**：utf8_next、utf8_prev、utf32_to_utf8、count_utf8_chars
- **字符边界遍历**：前向/后向移动 UTF-8 字符光标
- **编解码转换**：UTF-32 码点转 UTF-8 字节序列
- **字符计数**：统计 UTF-8 字节序列中的字符数
- **内联函数**：零开销，编译期优化

## 函数

### utf8_next()

```cpp
inline uint32_t utf8_next(const char* data, uint32_t pos, uint32_t end) noexcept
```

**描述**：向前移动一个 UTF-8 字符，返回下一个字符起始的字节偏移。

**参数**：
- `data`：UTF-8 字节序列
- `pos`：当前字符起始字节偏移
- `end`：字节序列末尾偏移（不含）

**返回**：下一个字符起始偏移；pos 已在末尾时返回 end

**特征**：
- 根据首字节识别 UTF-8 字符长度（1-4 字节）
- 安全边界检查，不会越界
- 内联函数，零开销

**实现逻辑**：
```cpp
// 1 字节：0xxxxxxx
// 2 字节：110xxxxx 10xxxxxx
// 3 字节：1110xxxx 10xxxxxx 10xxxxxx
// 4 字节：11110xxx 10xxxxxx 10xxxxxx 10xxxxxx

if      ((c & 0xF8u) == 0xF0u) { skip = 4u; }  // 11110xxx
else if ((c & 0xF0u) == 0xE0u) { skip = 3u; }  // 1110xxxx
else if ((c & 0xE0u) == 0xC0u) { skip = 2u; }  // 110xxxxx
else                            { skip = 1u; }  // 0xxxxxxx
```

**示例**：
```cpp
const char* text = "Hello世界";
uint32_t len = strlen(text);
uint32_t pos = 0;

// 遍历每个字符
while (pos < len) {
    uint32_t next = utf8_next(text, pos, len);
    // 处理字符 [pos, next)
    std::string_view ch(text + pos, next - pos);
    log("字符: {}", ch);
    pos = next;
}
```

### utf8_prev()

```cpp
inline uint32_t utf8_prev(const char* data, uint32_t pos) noexcept
```

**描述**：向后移动一个 UTF-8 字符，返回前一个字符起始的字节偏移。

**参数**：
- `data`：UTF-8 字节序列
- `pos`：当前位置（通常为光标位置）

**返回**：前一个字符起始偏移；pos 为 0 时返回 0

**特征**：
- 从 pos 向前查找，跳过连续字节（0x80–0xBF）
- 直到找到前导字节（非连续字节）
- 安全处理边界（pos 为 0）

**实现逻辑**：
```cpp
// 连续字节：10xxxxxx (0x80–0xBF)
// 跳过所有连续字节，找到前导字节

uint32_t p = pos - 1u;
while (p > 0u && (static_cast<uint8_t>(data[p]) & 0xC0u) == 0x80u) {
    --p;
}
return p;
```

**示例**：
```cpp
const char* text = "Hello世界";
uint32_t pos = strlen(text);  // 末尾

// 向后遍历每个字符
while (pos > 0) {
    uint32_t prev = utf8_prev(text, pos);
    // 处理字符 [prev, pos)
    std::string_view ch(text + prev, pos - prev);
    log("字符: {}", ch);
    pos = prev;
}
```

### utf32_to_utf8()

```cpp
inline uint32_t utf32_to_utf8(uint32_t cp, char out[4]) noexcept
```

**描述**：将 UTF-32 码点编码为 UTF-8 字节序列。

**参数**：
- `cp`：Unicode 码点（≤ 0x10FFFF）
- `out`：输出缓冲区（至少 4 字节）

**返回**：写入的字节数（1–4）

**特征**：
- 支持全部 Unicode 码点（U+0000 至 U+10FFFF）
- 无需 null 结尾
- 内联函数，编译期优化

**实现逻辑**：
```cpp
// 1 字节：U+0000 至 U+007F
if (cp < 0x80u) {
    out[0] = static_cast<char>(cp);
    return 1u;
}

// 2 字节：U+0080 至 U+07FF
if (cp < 0x800u) {
    out[0] = static_cast<char>(0xC0u | (cp >> 6));
    out[1] = static_cast<char>(0x80u | (cp & 0x3Fu));
    return 2u;
}

// 3 字节：U+0800 至 U+FFFF
if (cp < 0x10000u) {
    out[0] = static_cast<char>(0xE0u | (cp >> 12));
    out[1] = static_cast<char>(0x80u | ((cp >> 6) & 0x3Fu));
    out[2] = static_cast<char>(0x80u | (cp & 0x3Fu));
    return 3u;
}

// 4 字节：U+10000 至 U+10FFFF
out[0] = static_cast<char>(0xF0u | (cp >> 18));
out[1] = static_cast<char>(0x80u | ((cp >> 12) & 0x3Fu));
out[2] = static_cast<char>(0x80u | ((cp >> 6)  & 0x3Fu));
out[3] = static_cast<char>(0x80u | (cp         & 0x3Fu));
return 4u;
```

**示例**：
```cpp
// 编码单个码点
char buffer[4];
uint32_t len = utf32_to_utf8(U'世', buffer);
std::string_view utf8(buffer, len);
log("UTF-8: {}", utf8);  // "世"

// 编码多个码点
std::string result;
for (char32_t cp : U"Hello世界") {
    if (cp == 0) break;
    char buf[4];
    uint32_t n = utf32_to_utf8(cp, buf);
    result.append(buf, n);
}
log("结果: {}", result);  // "Hello世界"
```

### count_utf8_chars()

```cpp
inline uint32_t count_utf8_chars(const char* p, uint32_t len) noexcept
```

**描述**：统计 UTF-8 字节序列中的完整字符数。

**参数**：
- `p`：字节序列起始指针
- `len`：字节长度

**返回**：完整字符数（截断的不完整尾字节不计入）

**特征**：
- 统计完整字符（不计入截断的尾字节）
- 根据首字节识别字符长度
- 安全处理边界

**实现逻辑**：
```cpp
const char* end   = p + len;
uint32_t    count = 0;
while (p < end) {
    const auto c    = static_cast<uint8_t>(*p);
    uint32_t   step = (c < 0x80u) ? 1u      // ASCII
                    : (c < 0xE0u) ? 2u      // 2 字节
                    : (c < 0xF0u) ? 3u      // 3 字节
                    :               4u;     // 4 字节
    
    // 不完整字符（剩余字节不足）不计入
    if (static_cast<uint32_t>(end - p) < step) break;
    
    p += step;
    ++count;
}
return count;
```

**示例**：
```cpp
const char* text = "Hello世界";
uint32_t len = strlen(text);
uint32_t char_count = count_utf8_chars(text, len);
log("字符数: {}", char_count);  // 7

// 截断字符串
const char* truncated = "Hello世";  // "世"被截断（只有 2 字节）
uint32_t truncated_len = 8;
uint32_t truncated_count = count_utf8_chars(truncated, truncated_len);
log("字符数: {}", truncated_count);  // 5（不计入截断的"世"）
```

### 3. 遍历字符

```cpp
// 遍历文本中的每个字符
void foreach_char(const char* text, uint32_t len, std::function<void(std::string_view)> callback) {
    uint32_t pos = 0;
    while (pos < len) {
        uint32_t next = utf8_next(text, pos, len);
        std::string_view ch(text + pos, next - pos);
        callback(ch);
        pos = next;
    }
}

});
```

### 4. 删除字符（Backspace）

```cpp
// TextBox Backspace 删除前一个字符
void delete_prev_char() {
    if (cursor_pos_ == 0) return;
    
    const char* text = text_buffer_.data();
    uint32_t prev = utf8_prev(text, cursor_pos_);
    
    // 删除 [prev, cursor_pos_) 区间
    text_buffer_.erase(prev, cursor_pos_ - prev);
    cursor_pos_ = prev;
}
```

---

### 5. 删除字符（Delete）

```cpp
// TextBox Delete 删除后一个字符
void delete_next_char() {
    const char* text = text_buffer_.data();
    uint32_t len = text_buffer_.size();
    
    if (cursor_pos_ >= len) return;
    
    uint32_t next = utf8_next(text, cursor_pos_, len);
    
    // 删除 [cursor_pos_, next) 区间
    text_buffer_.erase(cursor_pos_, next - cursor_pos_);
}
```

### 8. 截断文本（按字符数）

```cpp
// 截断文本到指定字符数
std::string truncate_text(const char* text, uint32_t len, uint32_t max_chars) {
    uint32_t pos = 0;
    uint32_t count = 0;
    
    while (pos < len && count < max_chars) {
        pos = utf8_next(text, pos, len);
        ++count;
    }
    
    return std::string(text, pos);
}

// 使用
```

## 性能分析

### 内联优化

**特征**：
- 全部函数都是 `inline`
- 编译器可完全内联，零函数调用开销
- 常量传播优化

**示例**：
```cpp
// 编译器优化后等价于直接代码
uint32_t next = utf8_next(text, pos, len);
// → 内联为：
uint32_t c = static_cast<uint8_t>(text[pos]);
uint32_t skip = (c & 0xF8u) == 0xF0u ? 4u
              : (c & 0xF0u) == 0xE0u ? 3u
              : (c & 0xE0u) == 0xC0u ? 2u
              : 1u;
uint32_t next = pos + skip <= len ? pos + skip : len;
```

### 边界检查

**特征**：
- 全部函数都有边界检查
- 不会越界访问
- 安全处理空字符串

**示例**：
```cpp
// 空字符串
uint32_t next = utf8_next("", 0, 0);  // 返回 0

// pos 已在末尾
uint32_t next = utf8_next("Hello", 5, 5);  // 返回 5

// pos 为 0
uint32_t prev = utf8_prev("Hello", 0);  // 返回 0
```

## 最佳实践

### 1. 使用 utf8_next/prev 遍历

```cpp
// ✅ 推荐：使用 utf8_next 遍历
uint32_t pos = 0;
while (pos < len) {
    uint32_t next = utf8_next(text, pos, len);
    process_char(text + pos, next - pos);
    pos = next;
}

// ❌ 不推荐：按字节遍历
for (uint32_t i = 0; i < len; ++i) {
    // 可能在字符中间截断
}
```

```cpp
// ✅ 推荐：检查边界
if (cursor_pos_ < len) {
```

```cpp
uint32_t count = 0;
for (uint32_t pos = 0; pos < len; ) {
```

## 常见陷阱

### 1. 字节索引 vs 字符索引

```cpp
// ❌ 问题：混淆字节索引和字符索引
const char* text = "世界";
char ch = text[1];  // 不是第二个字符，而是"世"的第二个字节

// ✅ 解决：使用 utf8_next 获取字符边界
uint32_t pos = 0;
pos = utf8_next(text, pos, len);  // 跳过第一个字符
std::string_view second_char(text + pos, utf8_next(text, pos, len) - pos);
```

---

### 2. 截断字符

```cpp
// ❌ 问题：按字节截断可能在字符中间
std::string truncated(text, 5);  // 可能截断中文字符

// ✅ 解决：使用 utf8_next 找到字符边界
uint32_t pos = 0;
uint32_t count = 0;
while (pos < len && count < max_chars) {
    pos = utf8_next(text, pos, len);
    ++count;
}
std::string truncated(text, pos);
```

---

### 3. utf8_prev 越界

```cpp
// ❌ 问题：pos 为 0 时继续调用
uint32_t prev = utf8_prev(text, 0);
prev = utf8_prev(text, prev);  // 死循环

// ✅ 解决：检查边界
if (pos > 0) {
    prev = utf8_prev(text, pos);
}
```

---

### 4. count_utf8_chars 忽略截断字符

```cpp
// ❌ 问题：期望统计所有字节
const char* truncated = "Hello世";  // "世"被截断
uint32_t count = count_utf8_chars(truncated, 8);  // 返回 5，不是 6

// ✅ 理解：截断字符不计入
// count_utf8_chars 只统计完整字符，这是正确行为
```

## 完整示例

```cpp
#include <mine/text/Utf8.h>
#include <string>
#include <string_view>
#include <vector>

using namespace mine::text;

class TextEditor {
    std::string text_;
    uint32_t cursor_pos_ = 0;
    
public:
    // 插入字符
    void insert_char(char32_t cp) {
        char buffer[4];
        uint32_t len = utf32_to_utf8(cp, buffer);
        text_.insert(cursor_pos_, buffer, len);
        cursor_pos_ += len;
    }
    
    // 光标向右移动
    void move_right() {
        if (cursor_pos_ < text_.size()) {
            cursor_pos_ = utf8_next(text_.data(), cursor_pos_, text_.size());
        }
    }
    
    // 光标向左移动
    void move_left() {
        if (cursor_pos_ > 0) {
            cursor_pos_ = utf8_prev(text_.data(), cursor_pos_);
        }
    }
    
    // Backspace 删除
    void delete_prev() {
        if (cursor_pos_ == 0) return;
        uint32_t prev = utf8_prev(text_.data(), cursor_pos_);
        text_.erase(prev, cursor_pos_ - prev);
        cursor_pos_ = prev;
    }
    
    // Delete 删除
    void delete_next() {
        if (cursor_pos_ >= text_.size()) return;
        uint32_t next = utf8_next(text_.data(), cursor_pos_, text_.size());
        text_.erase(cursor_pos_, next - cursor_pos_);
    }
    
    // 获取字符数
    uint32_t char_count() const {
        return count_utf8_chars(text_.data(), text_.size());
    }
    
    // 获取所有字符
    std::vector<std::string_view> get_chars() const {
        std::vector<std::string_view> chars;
        const char* data = text_.data();
        uint32_t len = text_.size();
        uint32_t pos = 0;
        
        while (pos < len) {
            uint32_t next = utf8_next(data, pos, len);
            chars.emplace_back(data + pos, next - pos);
            pos = next;
        }
        
        return chars;
    }
    
    // 获取当前光标字符
    std::string_view current_char() const {
        if (cursor_pos_ >= text_.size()) return {};
        const char* data = text_.data();
        uint32_t next = utf8_next(data, cursor_pos_, text_.size());
        return {data + cursor_pos_, next - cursor_pos_};
    }
};

int main() {
    TextEditor editor;
    
    // 插入文本
    for (char32_t cp : U"Hello世界") {
        if (cp == 0) break;
        editor.insert_char(cp);
    }
    
    // 输出字符数
    log("字符数: {}", editor.char_count());  // 7
    
    // 输出所有字符
    for (auto ch : editor.get_chars()) {
        log("字符: {}", ch);
    }
    
    // 光标移动
    editor.move_right();
    log("当前字符: {}", editor.current_char());  // "e"
    
    // 删除字符
    editor.delete_prev();  // 删除 "H"
    log("字符数: {}", editor.char_count());  // 6
    
    return 0;
}
```

## 总结

`Utf8.h` 提供了 UTF-8 编解码与光标移动工具函数，具备：

1. **4 个工具函数**：utf8_next、utf8_prev、utf32_to_utf8、count_utf8_chars
2. **字符边界遍历**：前向/后向移动 UTF-8 字符光标
3. **编解码转换**：UTF-32 码点转 UTF-8 字节序列
4. **字符计数**：统计 UTF-8 字节序列中的字符数
5. **内联函数**：零开销，编译期优化

**使用建议**：
- 使用 `utf8_next`/`utf8_prev` 遍历字符
- 光标移动前检查边界
- 使用 `count_utf8_chars` 统计字符数
- UTF-32 转 UTF-8 预留 4 字节空间
- 避免混淆字节索引和字符索引
